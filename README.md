## Project Description
This project is a simple banking application that includes:
- Deposit and withdraw functions
- Interest calculation
- Loan eligibility check
- Fund transfer between accounts

The project includes **unit tests** for individual functions and **integration tests** for combined workflows like transfer + interest calculation.
